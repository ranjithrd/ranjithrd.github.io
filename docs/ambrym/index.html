Ambrym

Starter project with Pug and ExpressJS installed

Installation

git clone https://github.com/ranjithrd/ambrym.git PATH
cd PATH
rm -Rf .git
git init
yarn # OR npm i

Tech

Server: ExpressJS with CORS, ENV, BodyParser, DayJS configured Client: PugJS via Express with one default layout and paths for includes configured Debug: Nodemon and Prettier installed with yarn xxx commands added
