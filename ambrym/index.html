Ambrym

Starter project with Pug and ExpressJS installed
GitHub Docs (this page)

Installation

git clone https://github.com/ranjithrd/ambrym.git PATH
cd PATH
rm -Rf .git
git init
yarn # OR npm i

Tech

  • ExpressJS with CORS, ENV, BodyParser, DayJS configured
  • PugJS via Express with one default layout and paths for includes configured
  • Nodemon and Prettier installed with yarn xxx commands added
